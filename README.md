# Proyecto Ultra V4.5.2 — Equipo ordenado

Esta versión modifica únicamente la pantalla de equipo.

## Cambios
- Avatar central visible nuevamente.
- Slots organizados mediante una cuadrícula estable de cinco columnas.
- Ningún slot comparte posición con otro.
- Separación constante entre Cabeza, Cuello, Pecho, Brazos, Guantes, Túnica, Capa, Arma, Mano secundaria, Botas y anillos.
- Avatar separado de todos los recuadros.
- Anillos y botas subidos para evitar cortes.
- Inventario con scroll interno y última fila visible.
- Oro separado de la cuadrícula de objetos.
- Panel derecho compacto.
- Eliminación de información duplicada de velocidad de ataque.
- Ajustes especiales para iPhone horizontal con la barra de Safari visible.

Reemplaza todos los archivos. La caché del service worker es `ultra-v452`.
