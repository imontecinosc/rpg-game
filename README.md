# Proyecto Ultra V4.4 — Magia y HUD

## Cambios centrales
- Seis hechizos rápidos para todas las builds.
- Magia utilizable dentro y fuera del modo Batalla.
- Proyectiles guiados al objetivo seleccionado; el resultado se resuelve al impactar.
- Fatiga moderada por repetir el mismo hechizo.
- Cooldown mínimo entre magias diferentes y cooldown visual circular.
- Especiales 60/80 consumen maná y aguante.
- Autoataque arcano con grimorio o bastón.
- Nuevos pergaminos: Explosión, Incinerar, Témpano, Maldición, Veneno, Chispa, Rayo, Barrera arcana y Protección física.
- Daño en área según el perfil del arma.
- Buffs decimales entre 0,5% y 8%, con límites y presupuesto de poder.
- Velocidad de ataque visible en todas las armas.
- Ajustes simplificados: Gráficos, Audio y Partida.
- Venta por cantidad con slider, campo numérico y accesos rápidos.
- Retrato y oro en el HUD principal.
- Acciones de combate, curación, pociones y cambio Paz/Batalla sin detener el movimiento.
- Al entrar en Paz se elimina el objetivo.
- Sistema inicial de sonidos sintetizados.
- PWA instalable en iPhone, modo horizontal y soporte de altura visual real.

## Instalación
Sube todos los archivos de esta carpeta a la raíz de GitHub Pages. Para usarla como app en iPhone:
Safari → Compartir → Añadir a pantalla de inicio → Abrir como app web.
